Arduino Electrónica

- Resumen -

Durante esta práctica, diseñamos una serie de circuitos con el objetivo de comprender los fundamentos del funcionamiento de Arduino. Exploramos cómo el código se utiliza para proporcionar instrucciones a esta plataforma.

- Introducción -

Arduino es una plataforma de hardware y software de código abierto que permite crear proyectos electrónicos interactivos. Se basa en placas de circuito impresas con microcontroladores, que actúan como pequeños cerebros que controlan los componentes electrónicos. El software de Arduino es un entorno de desarrollo integrado (IDE) que facilita la programación en un lenguaje sencillo llamado C++.

- Materiales -

Los materiales utilizados para esta practica son:
- Una placa de Arduino Uno
- Un cable USB tipo B
- LEDS
- Un protoboard
- Resistencias
- Pulsador

- Desarrollo -

Durante el desarrollo de esta práctica, trabajamos con siete ejemplos distintos que el profesor nos proporcionó a través de la aplicación Tinkercad.

A continuación explicaremos cada uno de los ejemplos que se elaborarón en clases.

Desarrollo del ejemplo 1

Para el desarrollo de este ejemplo se armo un protoboard donde se conecto un LED, se conecto el lado negativo del LED a el negativo del protoboard, luego se conecto una resistencia del lado positivo del LED y del otro lado de la resistencia se conecto otro jumper que se conecta con el PIN 13 del Arduino. Despues se conecta el pulsador en medio del protoboard y se conecta el PIN izquierdo inferior con el PIN 8 del Arduino y en la misma columna se conecta una resistencia que va al negativo. Luego en el pin derecho inferior se conecta un jumper al positivo del protoboard. Finalmente se conecta el negativo del protoboard con Ground del Arduino y el positivo con el PIN de alimentación del Arduino.

- Conclusiones -

EEn este proyecto con Arduino Uno, exploramos el uso de entradas digitales mediante pulsadores y su interacción con LEDs. Aprendimos cómo configurar los pines digitales como entradas y cómo detectar cambios en el estado de los pulsadores.
