Arduino Electrónica

- Resumen -

En esta práctica, diseñamos circuitos para entender el control de servomotores con Arduino. Exploramos cómo el código puede mover el motor a ángulos específicos y cómo usar un potenciómetro para controlarlo manualmente.

- Introducción -

Un potenciómetro es un componente electrónico que funciona como un divisor de voltaje. Su principal uso es variar la resistencia en un circuito, permitiendo controlar manualmente dispositivos como servomotores. En esta práctica, utilizamos un potenciómetro para ajustar el ángulo de un servomotor, conectándolo a la plataforma Arduino para experimentar con el control de movimientos precisos. A continuación mostraremos algunos ejemplos del uso de servomotores explicados de manera básica.

- Conclusiones -

En conclusión, este proyecto nos permitió entender los fundamentos del control de servomotores utilizando Arduino.
